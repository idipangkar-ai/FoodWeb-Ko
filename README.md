````
## Getting Started

Clone the repo:

```bash
git clone https://github.com/your-username/food-order-app.git
cd food-order-app
````

````

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
````
